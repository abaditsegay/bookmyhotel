# Hotel Room Bulk Upload User Guide

## Quick Start

1. **Download Template** - Click "Download CSV Template" to get the sample file
2. **Fill Your Data** - Replace sample data with your actual room information  
3. **Upload File** - Select your completed CSV file
4. **Review & Import** - Check for errors and import rooms

## CSV Template Format

### Required Columns
- **Room Number**: Unique identifier (e.g., "101", "A1", "Presidential")
- **Room Type**: Category (Standard, Deluxe, Suite, Premium, Family, etc.)
- **Floor**: Floor number or level (1, 2, 3, etc.)
- **Capacity**: Maximum guests (1-20)
- **Base Price**: Nightly rate in your currency (numbers only)

### Optional Columns
- **Description**: Brief room description
- **Amenities**: Comma-separated list (WiFi, TV, Minibar, etc.)
- **Status**: Available, Occupied, Maintenance, Out of Order
- **Size (sqft)**: Room size in square feet
- **Bed Type**: Single, Double, Queen, King, Twin, etc.
- **View Type**: Garden, Ocean, City, Mountain, Pool, etc.
- **Has Balcony**: Yes or No
- **Has Kitchenette**: Yes or No
- **Accessibility Features**: Wheelchair accessible, etc.

## Tips for Success

✅ **Use the template** - Start with our sample file to ensure correct format
✅ **Unique room numbers** - Each room must have a different number
✅ **Valid data types** - Numbers for capacity and price, text for descriptions
✅ **Check spelling** - Room types should be consistent across your hotel
✅ **Test small batches** - Try uploading 5-10 rooms first

## Common Issues

❌ **Duplicate room numbers** - Make sure each room has a unique identifier
❌ **Invalid characters** - Avoid special characters in CSV data
❌ **Wrong file format** - Only CSV files are supported
❌ **Missing required fields** - All required columns must have values
❌ **Price format** - Use numbers only (e.g., 150, not $150)

## File Limits

- Maximum file size: 5MB
- Maximum rows: 1,000 rooms
- Supported format: CSV only

## Need Help?

If you encounter issues:
1. Download a fresh template
2. Check your data format matches the examples
3. Try uploading a smaller batch first
4. Contact support if problems persist
